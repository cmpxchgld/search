页内导航选项卡

你可以使用如下代码创建页内选项卡：

		<ul class="nav nav-tabs">
			<li class="active"><a href="#tab_course" data-toggle="tab">课程</a></li>
			<li><a href="#tab_teacher" data-toggle="tab">讲师</a></li>
			<li><a href="#tab_device" data-toggle="tab">设备</a></li>
			<li><a href="#tab_student" data-toggle="tab">学员</a></li>
		</ul>
		
		<div class="tab-content">
			<div class="tab-pane active" id="tab_course">
				<span class="nodata" id="no_course">本项目暂无相关课程</span>
				<div id="table_course_wrap" style="width:100%">
					<table id="table_course" style="height:180px">
					</table>
				</div>
			</div>
			<div class="tab-pane" id="tab_teacher">
				<span class="nodata" id="no_teacher">本项目暂无相关教师</span>
				<div id="table_teacher_wrap" style="width:100%">
					<table id="table_teacher" style="height:180px">
					</table>
				</div>
			</div>
			<div class="tab-pane" id="tab_device">
				<span class="nodata" id="no_equip">本项目暂无相关设备</span>
				<div id="table_equip_wrap" style="width:100%">
					<table id="table_equip" style="height:180px">
					</table>

  • nav-tabs和nav-justified样式一起使用可以让选项卡铺满宽度
  • 只需让a href指向目标div的ID并设置data-toggle="tab"即可完成选项卡切换绑定
  • li/a上均可以使用onclick绑定事件
  • 带有active样式的选项卡默认被激活
  • 可以通过调整.nav > li > a的样式（padding）来调整选项卡的高度和宽度

事件绑定

你可以使用如下代码对某个选项卡绑定切换事件：

		$("#tab_link_file").on('shown.bs.tab', function (e) {
			showProjAttchments();
		});

需要注意的是，如果在选项卡中使用datagrid，表格的初始化函数需要在事件中执行，这是因为被隐藏的选项卡无法获取尺寸，导致datagrid计算大小错误。
