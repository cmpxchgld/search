itcui_ext.js已经正式废弃，现在所有功能均集成在itcui.dev.js和itcui_frame.dev.js以及对应css中。

下载

目前UI组件库的最新版本是r4，包括EasyUI和Bootstrap的最新修改版，JS文件提供min和开发版，均未混淆。

JQuery

组件库中判断浏览器版本使用了$.support系列函数，因而需要JQuery1.9以上某些组件方能正常工作。

特别注意

  • 为了保护自己的权益，对于非集团内部的系统要使用压缩（必要时进行混淆）的代码。
  • 在使用DIV方式（EasyUI使用）的方式加载对话框时，必须避免重加载BS/EasyUI/JQuery的JS代码，否则容易出现各种难以解决的问题。
  • 在线发布的版本裁剪了BS/EasyUI的大量组件，如果需要可以自行下载完整版或者联系我们获得其他被裁剪组件的代码。
  • 如果某些组件的样式异常，请尝试在外层div加上样式bbox，因为现在border-box不是默认样式了

命名要求

大多数封装会根据用户提供的ID拼接出组件内各个元素，以加快选择器选择速度，如myid_title、myid_column等，所以不要使用会干扰jQuery选择器的字符串作为(field)ID和name，如#argvalue，myobject.property，this>element等。

链式调用

由于第一版封装比较仓促，暂时只保证可用性和最少的bug，目前版本不支持链式调用，即$(xxx).ITCUI_xxxx().attr(xxx)是行不通的。

分模块引用

你可以在自主打包版的source目录中找到每个模块的脚本和依存文件，或者修改package.py来选择所需的模块。为了使用打包脚本，你需要安装python2.7（python3.0不支持），并配置好环境变量，以便在任何目录下都能执行java.exe（用于压缩JS和CSS脚本。）

在package目录下执行python package.py可以生成一般页面的整合版，执行python package_frame.py可以生成框架版（包括导航树、导航选项卡以及依存代码）。

配置方法：

在package.py（package_frame.py的代码功能完全一样，只不过模块依存和最终生成文件不一样。）：

		#打包的输出路径
		JS_PATH = "js"
		IMG_PATH = "images"
		CSS_PATH = "css"
		selected_components = ["bs_core","bs_ext","bs_grid","bs_form","bs_button","bs_menu","itc_input","eu_datagrid","itc_pagination","itc_combo"];

三个常量定义了最终脚本、图片、样式表的输出文件，默认下会在当前目录下生成js、images、css三个目录（font目录不需要打包，也无法修改）。

selected_components里定义了要打包的组件，只要写出主组件的名字，依存的组件会自动打包进去。

默认情况下，打包会生成以下4个文件（以及images/font目录下的资源）：

  • js/itcui.dev.js
  • js/itcui.min.js
  • css/itcui.dev.css
  • css/itcui.min.css

jQuery不属于打包的对象，需要自行加载。

权限支持

通过为button,span,div添加privilege=xxx的参数，同时在页面的document.ready中执行priv()可以隐藏无权限的组件，而priv(true)则直接将无权限的组件移除。

开启权限支持必须定义全局变量privMapping，如果当前页面没有这个变量，则尝试去主框架中读取，否则权限控制无效。

建议在页面创建时就创建privMapping，不要使用异步方法，防止页面闪动。

例子

		<div class="btn-group btn-group-sm" id="edit" privilege="F-PROJ-EDIT">
	        <button type="button" class="btn btn-success dropdown-toggle" data-toggle="dropdown">
	      	填报
	        	<span class="caret"></span>
	        </button>
	        <ul class="dropdown-menu" id="menu_edit"></ul>
	    </div> 
	       
	    <div class="btn-group  btn-group-sm">
	        <button privilege="F-PROJ-EDIT" id="publish" type="button" class="btn btn-default" onclick="javascript:Project.publishProject('<%=id%>')">发布</button>
	        <button privilege="F-PROJ-EDIT" id="endproj" type="button" class="btn btn-default" onclick="javascript:Project.endProject('<%=id%>')">结项</button>
	        <div class="btn-group btn-group-sm">
				<button id="btn_moreopt" type="button" class="btn btn-default dropdown-toggle" data-toggle="dropdown">
			      	更多操作
			        <span class="caret"></span>	        
		        </button>
		        <ul class="dropdown-menu" id="menu_moreopt">	
			    </ul>
		    </div>
	    </div>
