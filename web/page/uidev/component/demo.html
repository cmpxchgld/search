关于演示页面

自r4版UI组件库发布后，为了避免对开发造成不必要的误导，本文档删除了所有v1.0版的演示页面。新版演示页面（和权限系统整合）请点击这里进入（会弹出新窗口）。

该页面中的选项卡和导航树配置来自TIMSS数据库，但是没拷贝对应页面，因此除了权限部分和UI演示页面，其余选项卡下的所有页面点击都是404错误

演示页面部分接口使用UI组件库在b5版中将发布的适配器编写，参数和入口更加简洁。

演示了什么？

  • EasyUI Layout
  • 与站点关联的数据权限
  • 树操作（动态载入），包括右键菜单、节点动态创建
  • Datagrid操作
  • 表单操作，包括创建、只读/编辑模式切换，表单验证，表单提交
