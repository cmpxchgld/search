创建一个单选/复选框

		<div class="row">
			<input type="checkbox" id="checkbox_1">
			<label for="checkbox_1">默认未选择的单选框</label>
			<input type="checkbox" checked id="checkbox_2">
			<label for="checkbox_2">默认已选择的单选框</label>
		</div>
		<div class="row">
			<input type="radio" name="iCheck" id="radio_1">
			<label for="radio_1">选项1</label>
			<input type="radio" name="iCheck" checked  id="radio_2">
			<label for="radio_2">选项2（默认选择）</label>
		</div>
	
		$(document).ready(function(e){
			//在实际写的时候请不要对所有input使用iCheck方法，因为该页面的input都是单选框，才可以这么用
			$('input').iCheck({
			    checkboxClass: 'icheckbox_flat-blue',
			    radioClass: 'iradio_flat-blue',
			});
		});

需要注意的是，在默认情况下，点击控件和控件的Label具有同等效果。

动态添加一个单选/复选框

单选/复选框的样式实际上是在document.ready事件触发时，调用iCheck方法应用样式的。如果需要动态添加控件，则需要手动调用该函数。

请注意，不要省略参数中指定的style（实际上可以通过修改参数在页面中实现不同样式的选择框）

		function addControl(){
			$("#row_controls").html("");
			$("#row_controls").append('<input type="checkbox" id="checkbox_3">')
			.append('<label for="checkbox_3">默认未选择的单选框</label>')
			.append('<input type="checkbox" checked id="checkbox_4">')
			.append('<label for="checkbox_4">默认已选择的单选框</label>');
		}

		function addControlStyle(){
			$("#checkbox_3").iCheck({
			    checkboxClass: 'icheckbox_flat-blue',
			    radioClass: 'iradio_flat-blue',
			});
			$("#checkbox_4").iCheck({
			    checkboxClass: 'icheckbox_flat-blue',
			    radioClass: 'iradio_flat-blue',
			});
		}
单击这里添加一些控件 单击这里为控件应用事件

获取控件的值

使用组件库提供的表单封装可以很方便的取值，但是取值只能判断$(input).parent().hasClass("checked")

赋值请使用$('input').iCheck('check')方法（取消使用uncheck），否则会导致组件行为异常。

动态调整控件的值

说明中的选择器可以随意替换。

		$('input').iCheck('check'); — 将输入框的状态设置为checked
		$('input').iCheck('uncheck'); — 移除 checked 状态
		$('input').iCheck('toggle'); — 切换选择状态
		$('input').iCheck('disable'); — 将输入框的状态设置为 disabled
		$('input').iCheck('enable'); — 移除 disabled 状态
		$('input').iCheck('update'); — 当通过参数修改控件样式时，可以使用该函数应用新的样式
		$('input').iCheck('destroy'); — 移除控件样式
